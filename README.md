# Python Quiz Engine

A command-line MCQ quiz system built using Python.

## Features
- JSON based question loading
- +4 / -1 scoring system
- 60 second timer per question
- Skip option
- Randomized questions
- 2x2 option layout
- Safe input handling
- Result summary
- Score saving to file

## Tech Stack
Python, JSON

## How to Run

```bash
python main.py
```
